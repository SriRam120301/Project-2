# 🤖 AI Studio Web Application

> An AI-powered web application built with **Google AI Studio (Gemini API)** and **Node.js**, designed to provide intelligent, real-time responses through a modern web interface.

---

# 📖 Project Overview

This project demonstrates the development of a web application powered by Google's Gemini Large Language Model.

The application provides users with an intuitive interface for interacting with generative AI while showcasing API integration, frontend development, and secure environment variable management.

This repository demonstrates practical experience in:

- AI Application Development
- REST API Integration
- Prompt Engineering
- Environment Configuration
- Modern JavaScript Development

---

# 🎯 Problem Statement

Many AI applications require complex setup and backend integration before they become usable.

This project simplifies AI interaction by creating a lightweight web application that connects directly with the Gemini API, allowing users to generate intelligent responses through a clean and responsive interface.

# ⚙️ Implementation

The application follows this workflow:

```
User Input
      │
      ▼
Frontend Interface
      │
      ▼
Gemini API Request
      │
      ▼
Google AI Studio
      │
      ▼
Generated Response
      │
      ▼
Display Response
```

---

# 💡 Key Design Decisions

This project was designed with the following principles:

- Keep the application lightweight.
- Separate configuration from application code.
- Store API keys securely using environment variables.
- Maintain a simple project structure for readability.
- Provide a responsive user experience.

# ⚠️ Limitations

Current limitations include:

- Requires an active Gemini API key.
- Internet connection is required.
- No user authentication.
- Conversation history is not permanently stored.
- Not optimized for production deployment.

Future versions may include authentication, database storage, streaming responses, and cloud deployment.

# 🚀 Getting Started

## Prerequisites

Before running the project, install:

- Node.js (v18 or later)
- npm
- Google Gemini API Key

---

## Clone the Repository

```bash
git clone https://github.com/SriRam120301/YOUR_REPOSITORY_NAME.git
```

---

## Navigate to the Project

```bash
cd YOUR_REPOSITORY_NAME
```

---

## Install Dependencies

```bash
npm install
```

---

## Configure Environment Variables

Create a file named:

```
.env.local
```

Add your Gemini API key:

```env
GEMINI_API_KEY=YOUR_API_KEY
```

---

## Start the Development Server

```bash
npm run dev
```

---

## Open the Application

Visit:

```
http://localhost:3000
```

(or the URL displayed in your terminal)
