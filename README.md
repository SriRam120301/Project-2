# 📌 Executive Summary

This project demonstrates the development of a modern AI-powered web application using **Google AI Studio (Gemini API)** and **Node.js**.

The application enables users to submit natural language prompts and receive AI-generated responses through a clean and responsive interface.

Beyond API integration, this project focuses on secure configuration management, modular application design, and building a reproducible AI application that follows modern software development practices.

---

# 🎯 Problem Statement

Generative AI models have become increasingly powerful, but integrating them into real-world web applications requires knowledge of API authentication, asynchronous programming, secure configuration, and user interface development.

The goal of this project was to design a lightweight AI application that demonstrates how a Large Language Model can be integrated into a modern web application while following industry best practices.

# 🎯 Key Design Decisions

Several design decisions were made throughout development:

- Environment variables were used to keep API keys secure.
- The project structure was kept modular for easier maintenance.
- Node.js was selected due to its lightweight ecosystem.
- Google Gemini API was chosen because of its ease of integration and strong generative capabilities.
- The application focuses on simplicity, allowing new developers to understand the complete workflow.

# ⚠️ Current Limitations

This project currently has several limitations:

- Conversation history is not stored permanently.
- No database integration.

# ⚡ Installation Guide

## Prerequisites

Before running this project, install:

- Node.js (v18 or later)
- npm
- Google Gemini API Key

---

## Clone Repository

```bash
git clone https://github.com/SriRam120301/YOUR_REPOSITORY_NAME.git
```

---

## Navigate to Project

```bash
cd YOUR_REPOSITORY_NAME
```

---

## Install Dependencies

```bash
npm install
```

---

## Configure Environment Variables

Create a file named:

```
.env.local
```

Add your Gemini API key:

```env
GEMINI_API_KEY=YOUR_API_KEY
```

---

## Run the Application

```bash
npm run dev
```

---

## Open in Browser

```
http://localhost:3000
```
